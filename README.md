# Проект пока сильно не доработан и пока не дорабатывается 
### что бы запустить проект используйте следующие команды
### (что бы проект работал необходимо также установить [python](https://www.python.org/downloads/) и пакетный менеджер pip | python -m pip install)

1. git clone https://github.com/Artyommmmmmmmmmmmmm/React-Django-project
2. cd React-Django-project
3. python -m venv venv
4. venv/Scripts/activate
5. pip install -r requirements.txt
6. cd backend
7. python manage.py runserver (эти команды запустили сервер следующие команды нужно выполнять в другой вкладке командной строки)
8. cd React-Django-project
9. cd frontend
10. npm i
11. npm start
